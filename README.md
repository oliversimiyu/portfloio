## Getting Started

1. **Clone or Download**: Get the project files to your local machine
2. **Open in Browser**: Simply open `index.html` in any modern web browser
3. **Local Server (Optional)**: For best experience, serve through a local web server

### Using a Local Server

You can use any of these methods to serve the website locally:

**Using Python (if installed):**
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

**Using Node.js (if installed):**
```bash
# Install live-server globally
npm install -g live-server

# Run in project directory
live-server
```

**Using VS Code:**
Install the "Live Server" extension and right-click on `index.html` to select "Open with Live Server"
